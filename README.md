# 🎙️ Vox Voice Assistant

Vox is an AI-powered voice assistant project designed to process voice commands and perform smart tasks through voice interaction. It aims to provide a simple and interactive assistant experience.

## ✨ Features

- 🎤 Voice command recognition
- 🤖 AI-powered responses
- ⚡ Fast and interactive communication
- 🔍 Smart task handling
- 🧠 User-friendly experience

## 🚀 Technologies Used

- Python
- Speech Recognition
- AI APIs / Libraries
- Text-to-Speech

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/irtzaahmad/vox-voice-assistant.git
```

Go to project folder:

```bash
cd vox-voice-assistant
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python main.py
```
